# chapter02/guessing_game

Simple `guess the number` game.

## Usage

Example run:

```console
$ cd chapter02/guessing_game
$ cargo run --release --quiet
Guess the number!
Please input your guess.
50
You guessed: 50
Too big!
Please input your guess.
25
You guessed: 25
Too big!
Please input your guess.
12
You guessed: 12
Too big!
Please input your guess.
6
You guessed: 6
Too Small!
Please input your guess.
9
You guessed: 9
You win!
```
